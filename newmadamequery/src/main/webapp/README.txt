*실행 주의사항*

*테스트 루틴*
1. 스터디 생성
상단의 STUDYGROUP -> 스터디등록 탭을 클릭하면 스터디를 생성할 수 있는 폼이 출력됩니다.
조건들을 입력하고 생성하기 버튼을 누르면 정상적으로 생성이 됩니다.
하지만 모집성별과 모집학년의 조건이 스터디를 등록하려는 사용자의 성별이나 학년 정보와 모순된다면, 스터디가 생성되지 않습니다.

위의 계정으로 로그인을 하셨다면 사용자의 성별은 여성이고 학년은 3학년입니다.
모집성별을 남성으로 택하거나 모집학년을 4학년이상으로 택하면 스터디생성이 불가합니다. 테스트 해보시길 바랍니다.

2. 스터디 검색
상단의 STUDYGROUP -> 스터디 검색 탭을 클릭하면 검색을 할 수 있는 폼이 출력됩니다. 조건을 입력하여 검색하기 버튼을 눌러보세요.

3. 스터디그룹 보기
상단의 STUDYGROUP -> 스터디그룹 보기 탭을 클릭하면 전체 스터디그룹 목록이 출력됩니다. 모집인원이 충족된 스터디그룹은 이 화면에 출력되지 않습니다.

4. 스터디그룹 팀원 신청하기
스터디 검색이나 스터디그룹 보기를 통하여 원하는 스터디그룹을 선택하면 '팀원 신청하기' 버튼을 눌러 진행할 수 있습니다. 
모집조건과 사용자의 정보가 일치하지 않는다면 신청이 거부됩니다.

위의 계정의 경우 3학년이므로 '졸업반 코테 스터디'에 신청해보시기 바랍니다. 학년 조건에 맞지 않아 가입 신청이 불가한 것을 확인하실 수 있습니다.

(예시) 사용자가 여성이지만 모집성별이 남성인 스터디그룹에 신청할 때, 사용자가 3학년이지만 모집학년이 4학년 이상인 스터디그룹에 신청할 때

5. 나의 신청현황
상단의 MYSTUDY -> 나의 신청현황을 클릭하면 내가 팀원으로 신청한 스터디그룹의 목록과 수락상태를 조회할 수 있습니다.
수락이 완료된 스터디그룹은 나의 스터디 보기에서 조회할 수 있습니다. 수락이 거절된 스터디그룹은 팀원으로 다시 신청할 수 없습니다.

6. 나의 스터디 보기
상단의 MYSTUDY -> 나의 스터디 보기를 클릭하면 내가 속해있는 스터디그룹의 목록이 출력됩니다.
탈퇴버튼을 누르면 곧바로 탈퇴가 진행됩니다. 신중히 클릭하여 주세요.
목록 중 그룹명을 택하면 스터디 그룹 멤버들과 소통할 수 있는 게시판이 출력됩니다.
글쓰기 버튼을 누르면 게시글을 작성할 수 있고, 게시글을 누르면 댓글을 작성하거나 삭제할 수 있습니다.
단, 내가 작성한 게시글이나 댓글에 대해서만 삭제가 가능합니다.

7. 스터디 그룹 관리 
상단의 MANAGE탭을 클릭하면 사용자가 관리하는 (팀장인) 스터디 그룹 목록이 출력됩니다. 그 중 한개를 클릭하면 스터디 정보와 신청자 정보를 조회할 수 있습니다. 

신청자 리스트에서 수락/거절 버튼을 눌러 테스트해보세요.
모집인원이 충족된 상태에서 신청자 수락을 누르면 경고창이 뜨면서 신청자 수락이 거부됩니다.

하단의 '정보수정'을 클릭하면 스터디그룹의 정보를 수정하는 폼이 출력됩니다.
스터디를 생성했을 때 모집성별과 모집학년을 '상관없음'으로 택했다면 다른 성별과 학년으로 수정할 수 없습니다.
모집성별이 특정성별(여성 혹은 남성)이었다면 그 특정성별의 반대성별로는 수정이 불가능합니다. (예시) 모집성별이 여성이었다면 남성으로 수정이 불가함
모집학년이 n학년 이상이었다면 n보다 큰 수로 변경이 불가능 합니다. (예시) 모집학년이 2학년 이상이었다면 4학년 이상으로 변경이 불가함
